Privacy Policy

Last updated: June 25, 2026

Motif is a Chrome extension that helps you hear French pronunciation, transcribe tab audio, and save vocabulary while reading the web.

Data Motif processes

When you use Motif, it may process:

  • Text you select on a web page
  • A screen region you choose for OCR
  • Audio from a tab you choose to transcribe
  • Words, translations, notes, and optional page context you save to your library (sentence snippet, page title, URL)

Where data goes

  • On your device: Speech synthesis, transcription, OCR, and translation (via Chrome’s built-in Translator API when available) run locally in your browser.
  • Hugging Face: On first use, Motif downloads machine learning model files from huggingface.co. No page content or audio is uploaded.
  • No Motif servers: Motif does not operate backend servers and does not collect analytics.

Data storage

Settings and saved vocabulary are stored in Chrome extension storage on your device. You can export or delete saved vocabulary from the Saved Library.
